《概要》
エンドロールと様々なMODを翻訳するlangファイルが入っています。
入っているものは非公式日本ユーザーフォーラムの
langファイルのトピックをご覧ください。


《動作確認バージョン》
非公式日本ユーザーフォーラムのlangファイルのトピックをご覧ください。


《MODの翻訳の導入方法》
MODフォルダ(.zip/.jar)の中の指定の場所にlangファイルを入れてください。
指定の場所については非公式日本ユーザーフォーラムの
langファイルのトピックをご覧ください。


《エンドロールの導入方法》
1.　credits.txtとend.txtを、使っているリソースパックの
　　「リソースパックのzip/assets/minecraft/texts」に置く。
　　リソースパックを作っていなかったらなかったら次へ進む。
　　！！ここでテキストの中身を見るとネタバレになるので
　　ネタバレしたくない人は開かないことに注意してください！！
2.　assetsというフォルダを作り、その中にminecraftというフォルダを作り、
　　またその中にtextsというフォルダを作る。
　　その中にcredits.txtとend.txtを入れる。
　　pack.mcmetaというファイルをassetsと同じ階層に作り
　　その中にテキスト形式で
　　{
  "pack": {
    "pack_format": 1,
    "description": "End_lang_JP"
  }
}
　　と書く。
3.　ゲーム内でリソースパックを選択して完了

詳細は非公式日本ユーザーフォーラムのlangファイルの
トピックをご覧ください。


《注意》



《報告・連絡》

非公式日本ユーザーフォーラムのlangファイルのトピックをご覧ください。
